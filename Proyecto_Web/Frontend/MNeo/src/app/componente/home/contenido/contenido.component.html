En esta version el Admin solo puede realizar activadades con Empleados y Conceptos



Empleados: CRUD Empleados

Concepto: Obtener lista de Conceptos

Jornada: No se Implemento en Front

RESUMEN DE LA PAGINA

Empleado Crear Empleado Me redirige a la ruta:'/crearEmpleado' donde a travez de un formulario ingreso y valido los datos de un empleado. Una vez pasado la validaciones se puede dar click al boton 'crearEmpleado' Luego de dar click, gracias a funcion crear() envio al back los datos del empleado Si no produce ningun error crea el empleado y luego me redirige a '/listarEmpleado' donde se ve la lista de todos los empleados En caso contrario se abre un DialogMat que muestra el error
Listar Empleado Muestra la lista de todos los empleados Ademas de listar se puede editar y borrar un empleado Editar: Se abre un DialogMat con los datos del empleado para que se modifique. Se realiza validacion en front y back si no produce error me redirige a pagina princial Borrar: Envio al back el id del empleado, en caso de no encontrar produce error .Una vez que encuentra el id lo borro y luego redirijo a pagina princial
Obtener Empleado me redirige a '/getEmpleado' y muestra un formulario para ingresar id y dar boton buscar Por la funcion editar() envio al back el id a buscar. Puede ocurrir 2 cosas: Ocurre error y me muestra DialogMat con el error Muestro los datos del Empleado Concepto Obtengo del Back la lista de todos los Conceptos Jornada Listar y Crear Jornada No esta desarrollado en el FrontEnd pero si se implemento en el Backend Boton de modoOscuro Al Realizar click aplica estilos distintos

LISTA DE ERRORES

Backend Empleado Crear Empleado La edad del empleado no puede ser menor a 18 años. Analiza la fecha de creacion y la fecha de nacimiento si la diferencia es menor a 18 entonces se produce el error Ya existe un empleado con el documento ingresado. Busco si el email del nuevo empleado existe en la base de datos en caso de ocurrir produce el error Ya existe un empleado con el email ingresado. La fecha de ingreso no puede ser posterior al día de la fecha. Recibo la fecha de ingreso y creacion si fechaIngreso > fechaCreacion produce error La fecha de nacimiento no puede ser posterior al día de la fecha. Recibo la fecha de nacimiento y creacion si fechaNacimiento > fechaCreacion produce error Actualizar Empleado No se encontró el empleado con Id Recibo un id y busco si existe en la base de datos de no encontrar produce el error Los mismos que en Crear Empleado Borrar Empleado No se encontró el empleado con Id Recibo un id y busco si existe en la base de datos de no encontrar produce el error Get Empleado No se encontró el empleado con Id Recibo un id y busco si existe en la base de datos de no encontrar produce el error Frontend Empleado Crear y Actualizar Empleado Nombre y Apellido 'campo' es requerido Si no ingresa nada produce el error Solo admite letras Si ingresa numeros o caracter especial produce el error FechaIngreso, FechaNacimiento,Dni 'campo' es requerido Email 'email' es requerido Formato email invalido Si no hay formato: _ '@' _ '.com' produce el error Obtener Empleado Id requerido Es necesario ingresar id para buscar empleado
